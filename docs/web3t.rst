=====
Web3t
=====

Web3t类封装了（兼容以太坊网络的）TrueChain网络相关的模块。

.. code-block:: javascript

  import Web3t from 'web3t';

Web3t对象的属性和Web3完全一致，如果对这个模块不了解，可以参考web3.js的官方文档。
